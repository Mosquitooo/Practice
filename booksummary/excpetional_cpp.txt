
=> iterator迭代器
	永远检查迭代器是否是一个有效值.

=> 尽可能的避免产生临时对象
	1. 使用const& 而不是 值传递
	2. 使用++i 而不是 i++
	3. 声明局部对象来存储返回值. 绝对不要返回临时对象的引用

=> 尽可能的使用标准库函数

=> 绝不允许异常从析构函数中渗透出来

=> 一般而言让构造函数成为explicit是好的，避免隐式转换.

=> 使用以下准则来确定一个运算子是成员还是非成员
	1. 一元运算符 重载为成员函数
	2. = () [] -> 必须是成员函数
	3. 赋值运算符 += -+ *= ... 必须是成员函数
	4. 其他所有二元运算符都必须是非成员函数

=> 总是在operator<< 和 operator>> 返回stream reference
=> 基类的析构函数必须成为虚函数
